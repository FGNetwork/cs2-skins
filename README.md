# FireGames Network CS2 Skins Library

## Install

```sh
  npm install @fgnetwork/cs2-skins
```
Or
```sh
  yarn add @fgnetwork/cs2-skins
```


### Usage

```ts
import { CS2Items } from "@fgnetwork/cs2-skins";

const items = CS2Items.getAll();

console.log(items);
```
